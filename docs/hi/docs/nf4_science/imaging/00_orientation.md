## GitHub Codespaces

GitHub Codespaces एनवायरनमेंट में इस ट्रेनिंग कोर्स को पूरा करने के लिए आवश्यक सभी सॉफ़्टवेयर, कोड और डेटा शामिल हैं, इसलिए तुम्हें खुद कुछ भी इंस्टॉल करने की ज़रूरत नहीं है।
हालांकि, लॉग इन करने के लिए तुम्हें एक (मुफ़्त) GitHub अकाउंट की आवश्यकता है, और यदि तुम इंटरफ़ेस से अपरिचित हो तो [GitHub Codespaces Orientation](../../envsetup/index.md) मिनी-कोर्स पूरा करके इससे परिचित होने के लिए कुछ मिनट लेना चाहिए।

## Docker इमेज पहले से डाउनलोड करना

एक बार जब तुमने अपना Codespace खोल लिया है, तो चलो इस ट्रेनिंग कोर्स के लिए आवश्यक सभी Docker इमेज पहले से डाउनलोड कर लें।
इससे बाद में समय की बचत होगी और वर्कफ़्लो का सुचारू निष्पादन सुनिश्चित होगा।

एक नया टर्मिनल टैब खोलो और निम्नलिखित कमांड चलाओ:

```bash
nextflow run nf-core/molkart -profile docker,test -stub -resume --outdir results
```

यह कमांड बैकग्राउंड में सभी आवश्यक Docker इमेज डाउनलोड करेगा।
तुम इसके चलते समय ओरिएंटेशन के बाकी हिस्से को जारी रख सकते हो।

!!!tip

    `-stub` फ़्लैग पाइपलाइन को वास्तविक डेटा प्रोसेस किए बिना तेज़ी से चलने की अनुमति देता है, जो इमेज डाउनलोड करने के लिए बिल्कुल सही है। तुम टर्मिनल टैब में प्रगति की निगरानी कर सकते हो।

## वर्किंग डायरेक्टरी

इस पूरे ट्रेनिंग कोर्स के दौरान, हम `nf4-science/imaging/` डायरेक्टरी में काम करेंगे।

टर्मिनल में यह कमांड चलाकर अभी डायरेक्टरी बदलो:

```bash
cd nf4-science/imaging/
```

!!!tip

    यदि किसी भी कारण से तुम इस डायरेक्टरी से बाहर चले जाते हो, तो तुम हमेशा इसमें वापस आने के लिए पूरा पाथ उपयोग कर सकते हो, यह मानते हुए कि तुम GitHub Codespaces ट्रेनिंग एनवायरनमेंट के भीतर इसे चला रहे हो:

    ```bash
    cd /workspaces/training/nf4-science/imaging
    ```

**अब, कोर्स शुरू करने के लिए, इस पेज के निचले दाएं कोने में तीर पर क्लिक करो।**
