# Nextflow for Science

ये कोर्स दिखाते हैं कि [Hello Nextflow](../hello_nextflow/) शुरुआती कोर्स में प्रस्तुत अवधारणाओं और घटकों को विशिष्ट वैज्ञानिक उपयोग के मामलों में कैसे लागू किया जाए। प्रत्येक कोर्स प्रशिक्षण मॉड्यूल की एक श्रृंखला है जो शिक्षार्थियों को उनके कौशल को क्रमिक रूप से विकसित करने में मदद करने के लिए डिज़ाइन की गई है।

!!! exercise "Nextflow for Genomics"

    !!! tip inline end ""

        :material-run-fast: Nextflow में जीनोमिक्स के लिए पाइपलाइन विकसित करना सीखें।

    यह कोर्स उन शोधकर्ताओं के लिए है जो अपनी खुद की जीनोमिक्स पाइपलाइन विकसित करना सीखना चाहते हैं। यह कोर्स एक सरल लेकिन कार्यात्मक जीनोमिक्स पाइपलाइन विकसित करने का तरीका दिखाने के लिए वेरिएंट कॉलिंग उपयोग के मामले का उपयोग करता है।

    [Nextflow for Genomics प्रशिक्षण शुरू करें :material-arrow-right:](genomics/){ .md-button .md-button--primary }

!!! exercise "Nextflow for RNAseq"

    !!! tip inline end ""

        :material-run-fast: Nextflow में RNAseq डेटा प्रोसेसिंग के लिए पाइपलाइन विकसित करना सीखें।

    यह कोर्स उन शोधकर्ताओं के लिए है जो अपनी खुद की RNAseq पाइपलाइन विकसित करना सीखना चाहते हैं। यह कोर्स एक सरल लेकिन कार्यात्मक RNAseq पाइपलाइन विकसित करने का तरीका दिखाने के लिए बल्क RNAseq प्रोसेसिंग उपयोग के मामले का उपयोग करता है।

    [Nextflow for RNAseq प्रशिक्षण शुरू करें :material-arrow-right:](rnaseq/){ .md-button .md-button--primary }

!!! exercise "Nextflow for Bioimaging"

    !!! tip inline end ""

        :material-run-fast: Nextflow में इमेजिंग डेटा के लिए पाइपलाइन चलाना सीखें।

    यह कोर्स उन शोधकर्ताओं के लिए है जो बायोइमेजिंग पाइपलाइन चलाना और कॉन्फ़िगर करना सीखना चाहते हैं। यह कोर्स किसी भी पाइपलाइन पर लागू होने वाले आवश्यक Nextflow उपयोग पैटर्न को प्रदर्शित करने के लिए nf-core/molkart का उपयोग करता है।

    [Nextflow for Bioimaging प्रशिक्षण शुरू करें :material-arrow-right:](imaging/){ .md-button .md-button--primary }

कम्युनिटी फोरम के [Training सेक्शन](https://community.seqera.io/c/training/) में पोस्ट करके हमें बताएं कि तुम यहां और कौन से डोमेन और उपयोग के मामले देखना चाहते हो।
