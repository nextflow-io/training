# Nextflow Run

**Nextflow Run पुनरुत्पादनीय और स्केलेबल डेटा विश्लेषण वर्कफ़्लो चलाने के लिए एक व्यावहारिक परिचय है।**

व्यावहारिक उदाहरणों और निर्देशित अभ्यासों के माध्यम से काम करते हुए, तुम Nextflow का उपयोग करने की बुनियादी बातें सीखोगे, जिसमें पाइपलाइनों को निष्पादित करना, फ़ाइलों और सॉफ़्टवेयर निर्भरताओं को प्रबंधित करना, निष्पादन को सहजता से समानांतर करना, और विभिन्न कंप्यूटिंग वातावरणों में वर्कफ़्लो चलाना शामिल है।

तुम Nextflow के साथ वर्कफ़्लो चलाना शुरू करने के लिए कौशल और आत्मविश्वास प्राप्त करोगे।

## कोर्स अवलोकन

### तुम क्या करोगे

यह कोर्स व्यावहारिक है, जिसमें लक्ष्य-उन्मुख अभ्यास हैं जो जानकारी को धीरे-धीरे पेश करने के लिए संरचित हैं।

तुम एक Nextflow पाइपलाइन के कई संस्करणों को निष्पादित करोगे जो टेक्स्ट इनपुट को प्रोसेस करती है।
तुम एक सरल संस्करण से शुरू करोगे जिसमें एक ही चरण होता है, और अंततः एक बहु-चरणीय संस्करण तक प्रगति करोगे जो टेबुलर टेक्स्ट इनपुट की एक CSV फ़ाइल लेता है, कुछ रूपांतरण चरण चलाता है, और एक एकल टेक्स्ट फ़ाइल आउटपुट करता है जिसमें एक ASCII चित्र होता है जिसमें एक पात्र रूपांतरित टेक्स्ट बोल रहा है।

यह कोर्स पाइपलाइनों को चलाने पर केंद्रित है (मुख्य `nextflow run` कमांड के नाम पर)।
यदि तुम Nextflow पाइपलाइनों को विकसित करने के परिचय की तलाश में हो, तो [Hello Nextflow](../hello_nextflow/index.md) देखो।
