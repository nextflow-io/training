# अभिविन्यास

GitHub Codespaces वातावरण में इस प्रशिक्षण कोर्स को पूरा करने के लिए आवश्यक सभी सॉफ़्टवेयर, कोड और डेटा शामिल हैं, इसलिए तुम्हें खुद कुछ भी इंस्टॉल करने की ज़रूरत नहीं है।
हालांकि, लॉग इन करने के लिए तुम्हें एक (मुफ़्त) अकाउंट की आवश्यकता है, और तुम्हें इंटरफ़ेस से परिचित होने के लिए कुछ मिनट लेने चाहिए।

यदि तुमने अभी तक ऐसा नहीं किया है, तो कृपया आगे बढ़ने से पहले [इस लिंक](../../envsetup/) को फ़ॉलो करो।

## प्रदान की गई सामग्री

इस प्रशिक्षण कोर्स के दौरान, हम `side-quests/` डायरेक्टरी में काम करेंगे।
इस डायरेक्टरी में वे सभी कोड फ़ाइलें, टेस्ट डेटा और सहायक फ़ाइलें हैं जिनकी तुम्हें आवश्यकता होगी।

इस डायरेक्टरी की सामग्री को एक्सप्लोर करने के लिए स्वतंत्र महसूस करो; ऐसा करने का सबसे आसान तरीका GitHub Codespaces वर्कस्पेस के बाईं ओर फ़ाइल एक्सप्लोरर का उपयोग करना है।
वैकल्पिक रूप से, तुम `tree` कमांड का उपयोग कर सकते हो।
कोर्स के दौरान, हम डायरेक्टरी संरचना और सामग्री को पठनीय रूप में प्रस्तुत करने के लिए `tree` के आउटपुट का उपयोग करते हैं, कभी-कभी स्पष्टता के लिए मामूली संशोधनों के साथ।

यहां हम दूसरे स्तर तक सामग्री की तालिका बनाते हैं:

```bash
tree . -L 2
```

यदि तुम इसे `side-quests` के अंदर चलाते हो, तो तुम्हें निम्नलिखित आउटपुट दिखाई देना चाहिए:

```console title="Directory contents"
.
├── metadata
├── nf-core
├── nf-test
├── solutions
├── splitting_and_grouping
└── workflows_of_workflows
```

**शुरू करने के लिए तुम्हें यह जानना चाहिए:**

- **प्रत्येक डायरेक्टरी एक व्यक्तिगत साइड क्वेस्ट से संबंधित है।**
  उनकी सामग्री संबंधित साइड क्वेस्ट के पेज पर विस्तृत है।

- **`solutions` डायरेक्टरी** में पूर्ण वर्कफ़्लो और/या मॉड्यूल स्क्रिप्ट शामिल हैं जो प्रत्येक साइड क्वेस्ट के विभिन्न चरणों को पूरा करने से प्राप्त होते हैं।
  इनका उद्देश्य तुम्हारे काम की जांच करने और किसी भी समस्या का निवारण करने के लिए संदर्भ के रूप में उपयोग किया जाना है।

!!!tip "सुझाव"

    यदि किसी कारण से तुम इस डायरेक्टरी से बाहर चले जाते हो, तो तुम इसमें वापस आने के लिए हमेशा यह कमांड चला सकते हो:

    ```bash
    cd /workspaces/training/side-quests
    ```

अब, कोर्स शुरू करने के लिए, इस पेज के निचले दाएं कोने में तीर पर क्लिक करो।
