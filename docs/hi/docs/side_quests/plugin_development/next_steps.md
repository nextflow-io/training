## 1. अपनी plugin यात्रा जारी रखने के शीर्ष 3 तरीके

आगे क्या करना है, इसके लिए यहाँ तीन सुझाव दिए गए हैं।

### 1.1. Plugin registry एक्सप्लोर करो

[Nextflow Plugin Registry](https://registry.nextflow.io/) ब्राउज़ करो और देखो कि कौन-कौन से plugins उपलब्ध हैं।
तुम्हें इनपुट वैलिडेशन, क्लाउड प्लेटफ़ॉर्म इंटीग्रेशन, नोटिफ़िकेशन, provenance ट्रैकिंग और बहुत कुछ के लिए plugins मिलेंगे।

### 1.2. मौजूदा plugin का सोर्स कोड पढ़ो

लोकप्रिय plugins का सोर्स कोड देखो और समझो कि वे कैसे बनाए गए हैं।
[nf-hello](https://github.com/nextflow-io/nf-hello) plugin जानबूझकर सरल और अच्छी तरह से डॉक्युमेंट किया गया है।
[official plugins repository](https://github.com/nextflow-io/plugins) में executor और filesystem इम्प्लीमेंटेशन सहित और जटिल उदाहरण मौजूद हैं।

### 1.3. कुछ उपयोगी बनाओ

सोचो कि Nextflow में तुम कौन-सी ऐसी कार्यक्षमता चाहते थे जो तुम्हें नहीं मिली।
कुछ विचार:

- अपने संगठन के लिए कस्टम लॉगिंग या रिपोर्टिंग
- आंतरिक टूल्स या APIs के साथ इंटीग्रेशन
- डोमेन-विशिष्ट यूटिलिटी फ़ंक्शन
- अपने पसंदीदा प्लेटफ़ॉर्म पर नोटिफ़िकेशन

छोटे से शुरू करो, उसे काम करवाओ, फिर धीरे-धीरे बेहतर बनाओ।

## 2. कम्युनिटी से मदद लो

- [Nextflow Slack](https://www.nextflow.io/slack-invite.html): सवाल पूछो और अपना काम शेयर करो
- [Community forum](https://community.seqera.io/): विचार-विमर्श करो और सलाह लो
- [GitHub discussions](https://github.com/nextflow-io/nextflow/discussions): तकनीकी सवाल और फ़ीचर रिक्वेस्ट

अगर तुम कोई उपयोगी plugin बनाते हो, तो उसे plugin registry के ज़रिए कम्युनिटी के साथ शेयर करने पर विचार करो।
