# मदद प्राप्त करना

चाहे तुम्हें शुरू करने में कठिनाई हो रही हो, बीच में अटक गए हो या कोई फॉलो-अप प्रश्न हो, संपर्क करने में संकोच मत करो! हमारी कम्युनिटी टीम मदद के लिए यहाँ है, और Nextflow कम्युनिटी बहुत सक्रिय, समावेशी और मदद करने के लिए उत्सुक है।

यहाँ मुख्य विकल्प हैं जो तुम्हारी आवश्यकता के अनुसार उपलब्ध हैं।

<div class="grid cards" markdown>

-   :material-forum-outline:{ .lg .middle } __कम्युनिटी फ़ोरम__

    ---

    हमारे कम्युनिटी फ़ोरम में प्रशिक्षण के लिए समर्पित एक श्रेणी है, जो प्रश्न पोस्ट करने या प्रशिक्षण के साथ आने वाली किसी भी समस्या की रिपोर्ट करने के लिए एक बेहतरीन जगह है। हो सकता है कि तुम्हारा प्रश्न पहले ही पूछा जा चुका हो --और उसका उत्तर भी मिल चुका हो!

    [प्रशिक्षण फ़ोरम में शामिल हों:material-arrow-right:](https://community.seqera.io/c/training/39){ .md-button .md-button--primary .mt-1 }

-   :material-slack:{ .lg .middle } __Slack चैनल__

    ---

    अगर तुम Slack पर हो, तो Nextflow Slack वर्कस्पेस में प्रशिक्षण चैनल में हमसे संपर्क करने के लिए स्वागत है। Nextflow Slack साथी डेवलपर्स के साथ चैट करने और सामान्य रूप से Nextflow कम्युनिटी के साथ जुड़ने के लिए एक शानदार जगह है।

    [Nextflow Slack में शामिल हों :material-arrow-right:](https://www.nextflow.io/slack-invite.html){ .md-button .md-button--primary .mt-1 }

-   :material-github:{ .lg .middle } __Github issues__

    ---

    अगर तुम्हें प्रशिक्षण सामग्री में कोई त्रुटि दिखाई देती है, तो कृपया Github रिपॉज़िटरी में एक issue खोलकर इसकी रिपोर्ट करो। चाहे वह टाइपो हो, फ़ॉर्मेटिंग समस्या हो या कोड को प्रभावित करने वाला वास्तविक बग हो, हमें बताओ ताकि हम इसे ठीक कर सकें! हम सीधे PR का भी स्वागत करते हैं।

    [एक issue रिपोर्ट करें:material-arrow-right:](https://github.com/nextflow-io/training/issues){ .md-button .md-button--primary .mt-1 }

-   :octicons-comment-ai-16:{ .lg .middle } __Seqera AI असिस्टेंट__

    ---

    Seqera AI एक AI असिस्टेंट है जो Nextflow और nf-core संसाधनों पर प्रशिक्षित है। यह तुम्हारे कोड को डीबग करने, Nextflow अवधारणाओं को स्पष्ट करने और दस्तावेज़ीकरण को तेज़ी से खोजने में मदद कर सकता है। इसे 24/7 उपलब्ध ट्यूटर के रूप में सोचो जब तुम कोर्स के माध्यम से काम कर रहे हो।

    [Seqera AI से पूछें:material-arrow-right:](https://github.com/nextflow-io/training/issues){ .md-button .md-button--primary .mt-1 }

-   :material-book-open-variant:{ .lg .middle } __Nextflow दस्तावेज़ीकरण__

    ---

    आधिकारिक दस्तावेज़ीकरण सभी भाषा सुविधाओं और कॉन्फ़िगरेशन विकल्पों के लिए अंतिम गाइड है। विशिष्ट विषयों में गहराई से जाने, उन्नत सुविधाओं का पता लगाने और विस्तृत सिंटैक्स संदर्भ खोजने के लिए इस प्रशिक्षण के साथ इसका उपयोग करो।

    [दस्तावेज़ ब्राउज़ करें:material-arrow-right:](https://www.nextflow.io/docs/latest){ .md-button .md-button--primary .mt-1 }

-   :material-account-hard-hat:{ .lg .middle } __व्यावसायिक सहायता__

    ---

    Nextflow एक मुफ़्त, ओपन-सोर्स सॉफ़्टवेयर है जिसे [Seqera](https://seqera.io/) द्वारा विकसित किया गया है, जो स्पेन में मुख्यालय वाली एक कंपनी है जिसके UK और US में सैटेलाइट ऑफ़िस हैं। हम Nextflow के लिए व्यावसायिक सहायता सेवाएँ प्रदान करते हैं, जिसमें कस्टम प्रशिक्षण शामिल है।

    [संपर्क करें:material-arrow-right:](https://seqera.io/demo/){ .md-button .md-button--primary .mt-1 }
