# मैनुअल इंस्टॉलेशन

अपने लोकल एनवायरनमेंट में ट्रेनिंग चलाने के लिए ज़रूरी सब कुछ मैनुअली इंस्टॉल करना संभव है।

यहाँ हमने स्टैंडर्ड POSIX-कम्पैटिबल सिस्टम्स (जैसे लैपटॉप जैसी पर्सनल मशीन) पर ऐसा करने का तरीका डॉक्यूमेंट किया है।
ध्यान रखो कि तुम्हारे स्पेसिफिक सिस्टम के आधार पर कुछ डिटेल्स अलग हो सकती हैं।

!!! tip "सुझाव"

    आगे बढ़ने से पहले, क्या तुमने [Devcontainers अप्रोच](03_devcontainer.md) का उपयोग करने पर विचार किया है?
    यह मैनुअल इंस्टॉलेशन की ज़रूरत के बिना सभी आवश्यक टूल्स और डिपेंडेंसीज़ प्रदान करता है।

## सामान्य सॉफ़्टवेयर आवश्यकताएँ

Nextflow को किसी भी POSIX-कम्पैटिबल सिस्टम (Linux, macOS, Windows Subsystem for Linux, आदि) पर Java इंस्टॉल होने के साथ उपयोग किया जा सकता है।
हमारे ट्रेनिंग कोर्सेज़ की कुछ अतिरिक्त आवश्यकताएँ हैं।

कुल मिलाकर, तुम्हें निम्नलिखित सॉफ़्टवेयर इंस्टॉल करने होंगे:

- Bash या समकक्ष शेल
- [Java 11 (या बाद का, 21 तक)](https://www.oracle.com/technetwork/java/javase/downloads/index.html)
- [Git](https://git-scm.com/)
- [Docker](https://docs.docker.com/get-docker/)
- [Conda](https://conda.io/) 4.5 (या बाद का)
- [VSCode](https://code.visualstudio.com) [Nextflow एक्सटेंशन](https://www.nextflow.io/docs/latest/developer-env.html#devenv-nextflow) के साथ

VSCode एप्लिकेशन तकनीकी रूप से ऑप्शनल है लेकिन हम दृढ़ता से सुझाव देते हैं कि तुम इसे कोर्सेज़ के साथ काम करने के लिए और सामान्य रूप से अपने Nextflow डेवलपमेंट कार्य के लिए उपयोग करो।

Nextflow डॉक्यूमेंटेशन मैनुअल [Environment setup](https://www.nextflow.io/docs/latest/developer-env.html) के तहत इन डिपेंडेंसीज़ को इंस्टॉल करने के निर्देश प्रदान करता है।

## Nextflow और nf-core टूल्स

तुम्हें Nextflow खुद को इंस्टॉल करना होगा, साथ ही nf-core टूल्स भी, जैसा कि नीचे लिंक किए गए आर्टिकल्स में विस्तार से बताया गया है:

- [Nextflow इंस्टॉलेशन](https://www.nextflow.io/docs/latest/install.html)
- [nf-core टूल्स](https://nf-co.re/docs/nf-core-tools/installation)

हम Nextflow के लिए सेल्फ-इंस्टॉल ऑप्शन और nf-core टूल्स के लिए PyPI ऑप्शन का उपयोग करने की सिफारिश करते हैं।

!!! warning "चेतावनी: वर्जन कम्पैटिबिलिटी"

    <!-- Any update to this content needs to be copied to the home page -->
    **जनवरी 2026 से, हमारे सभी Nextflow ट्रेनिंग कोर्सेज़ को Nextflow वर्जन 25.10.2 या बाद के वर्जन की ज़रूरत है, strict v2 सिंटैक्स एक्टिवेट होने के साथ, जब तक अन्यथा नोट न किया गया हो।**

    वर्जन आवश्यकताओं और strict v2 सिंटैक्स के बारे में अधिक जानकारी के लिए, कृपया [Nextflow versions](../info/nxf_versions.md) गाइड देखो।

    पुराने सिंटैक्स के अनुरूप ट्रेनिंग मटेरियल के पुराने वर्जन इस वेबपेज के मेनू बार में वर्जन सिलेक्टर के माध्यम से उपलब्ध हैं।

## ट्रेनिंग मटेरियल्स

ट्रेनिंग मटेरियल्स डाउनलोड करने का सबसे आसान तरीका इस कमांड का उपयोग करके पूरी रिपॉज़िटरी को क्लोन करना है:

```bash
git clone https://github.com/nextflow-io/training.git
```

प्रत्येक कोर्स की अपनी डायरेक्टरी है।
किसी कोर्स के साथ काम करने के लिए, एक टर्मिनल विंडो खोलो (आदर्श रूप से, VSCode एप्लिकेशन के अंदर से) और संबंधित डायरेक्टरी में `cd` करो।

फिर तुम वेबसाइट पर दिए गए कोर्स निर्देशों का पालन कर सकते हो।
