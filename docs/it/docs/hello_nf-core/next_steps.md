## Il vostro percorso

Avete iniziato imparando a recuperare ed eseguire una pipeline demo, poi avete affrontato la conversione di un semplice flusso di lavoro Nextflow in una pipeline nf-core.
Avete imparato come creare uno scaffold di pipeline utilizzando un template e avete innestato la pipeline esistente su quello scaffold.
Poi avete gradualmente perfezionato la pipeline sostituendo uno dei moduli locali con un modulo nf-core, trasformato un altro modulo locale per adattarlo agli standard nf-core e aggiunto la validazione dell'input.

### Cosa avete costruito

La vostra pipeline finale `core-hello` ora ha:

- **Struttura standardizzata** utilizzando il template nf-core con directory organizzate per workflow, subworkflow, moduli e configurazione
- **Moduli della community** dal repository nf-core (`cat/cat`) insieme ai vostri moduli personalizzati
- **Validazione completa** che controlla sia i parametri che i dati di input prima dell'esecuzione della pipeline
- **Configurazione professionale** con profili per diversi ambienti di esecuzione
- **Documentazione completa** e metadati che seguono le convenzioni nf-core

### Competenze chiave acquisite

Attraverso questo corso pratico, avete imparato a:

1. **Navigare e comprendere** la struttura delle pipeline nf-core esplorando una pipeline esistente
2. **Ristrutturare i flussi di lavoro** per renderli componibili e adattarli al template nf-core
3. **Trovare e integrare** moduli pre-costruiti dal repository della community
4. **Creare moduli personalizzati** seguendo gli standard nf-core per denominazione, struttura e metadati
5. **Implementare la validazione** utilizzando nf-schema per individuare errori in anticipo con feedback chiari

Ora siete equipaggiati con le conoscenze fondamentali per costruire pipeline nf-core pronte per la produzione che seguono le best practice della community.

## Prossimi passi per sviluppare le vostre competenze

Ecco i nostri 3 suggerimenti principali per cosa fare dopo:

- Applicate Nextflow a un caso d'uso di analisi scientifica con [Nextflow for Science](../nf4_science/index.md)
- Esplorate funzionalità più avanzate di Nextflow con le [Side Quests](../side_quests/index.md)
- Coinvolgetevi [unendovi alla community nf-core](https://nf-co.re/join).

Infine, vi consigliamo di dare un'occhiata a [**Seqera Platform**](https://seqera.io/), una piattaforma basata su cloud sviluppata dai creatori di Nextflow che rende ancora più facile lanciare e gestire i vostri flussi di lavoro, oltre a gestire i vostri dati ed eseguire analisi in modo interattivo in qualsiasi ambiente.
