## Il vostro percorso

Avete iniziato imparando a recuperare ed eseguire una pipeline demo, poi avete affrontato la conversione di un semplice workflow Nextflow in una pipeline nf-core.
Avete imparato a creare uno scaffold di pipeline utilizzando un template e avete innestato la pipeline esistente su quello scaffold.
Poi avete gradualmente perfezionato la pipeline sostituendo uno dei moduli locali con un modulo nf-core, avete trasformato un altro modulo locale per adattarlo agli standard nf-core e avete aggiunto la validazione dell'input.

### Cosa avete costruito

La vostra pipeline finale `core-hello` ora dispone di:

- **Struttura standardizzata** utilizzando il template nf-core con directory organizzate per workflows, subworkflows, moduli e configurazione
- **Moduli della community** dal repository nf-core (`cat/cat`) insieme ai vostri moduli personalizzati
- **Validazione completa** che verifica sia i parametri che i dati di input prima dell'esecuzione della pipeline
- **Configurazione professionale** con profili per diversi ambienti di esecuzione
- **Documentazione completa** e metadati seguendo le convenzioni nf-core

### Competenze chiave acquisite

Attraverso questo corso pratico, avete imparato a:

1. **Navigare e comprendere** la struttura delle pipeline nf-core esplorando una pipeline esistente
2. **Ristrutturare i workflow** per renderli componibili e adattarli al template nf-core
3. **Trovare e integrare** moduli pre-costruiti dal repository della community
4. **Creare moduli personalizzati** seguendo gli standard nf-core per denominazione, struttura e metadati
5. **Implementare la validazione** utilizzando nf-schema per individuare errori precocemente con feedback chiari

Siete ora equipaggiati con le conoscenze fondamentali per costruire pipeline nf-core pronte per la produzione che seguono le best practice della community.

Ecco i nostri 3 suggerimenti principali per i prossimi passi:
