# Ottenere aiuto

Che abbiate difficoltà a iniziare, vi siate bloccati a metà strada o abbiate domande di approfondimento, non esitate a contattarci! Il nostro team della community è qui per aiutarvi, e la community Nextflow in generale è molto attiva, inclusiva e desiderosa di aiutare.

Ecco le principali opzioni disponibili a seconda di ciò che state cercando.

<div class="grid cards" markdown>

-   :material-forum-outline:{ .lg .middle } __Forum della community__

    ---

    Il nostro forum della community ha una categoria dedicata alla formazione, che è un ottimo posto per pubblicare domande o segnalare eventuali problemi che potreste avere con la formazione. Potreste persino scoprire che la vostra domanda è già stata posta --e risposta!

    [Unitevi al forum della formazione:material-arrow-right:](https://community.seqera.io/c/training/39){ .md-button .md-button--primary .mt-1 }

-   :material-slack:{ .lg .middle } __Canale Slack__

    ---

    Se usate Slack, siete i benvenuti a contattarci nel canale della formazione nello spazio di lavoro Slack di Nextflow. Lo Slack di Nextflow è un ottimo posto per chattare con altri sviluppatori e interagire con la community Nextflow in generale.

    [Unitevi allo Slack di Nextflow :material-arrow-right:](https://www.nextflow.io/slack-invite.html){ .md-button .md-button--primary .mt-1 }

-   :material-github:{ .lg .middle } __Issue su Github__

    ---

    Se individuate un errore nei materiali di formazione, segnalatelo aprendo un'issue nel repository Github. Che si tratti di un errore di battitura, un problema di formattazione o un vero e proprio bug che influisce sul codice, fatecelo sapere così possiamo correggerlo! Accogliamo volentieri anche PR dirette.

    [Segnala un problema:material-arrow-right:](https://github.com/nextflow-io/training/issues){ .md-button .md-button--primary .mt-1 }

-   :octicons-comment-ai-16:{ .lg .middle } __Assistente AI di Seqera__

    ---

    Seqera AI è un assistente AI addestrato sulle risorse Nextflow e nf-core. Può aiutarvi a fare il debug del vostro codice, chiarire concetti di Nextflow e cercare la documentazione più velocemente. Pensatelo come avere un tutor disponibile 24/7 mentre lavorate sui corsi.

    [Chiedi a Seqera AI:material-arrow-right:](https://github.com/nextflow-io/training/issues){ .md-button .md-button--primary .mt-1 }

-   :material-book-open-variant:{ .lg .middle } __Documentazione Nextflow__

    ---

    La documentazione ufficiale è la guida definitiva a tutte le funzionalità del linguaggio e alle opzioni di configurazione. Usatela insieme a questa formazione per approfondire argomenti specifici, esplorare funzionalità avanzate e trovare riferimenti dettagliati sulla sintassi.

    [Sfoglia la documentazione:material-arrow-right:](https://www.nextflow.io/docs/latest){ .md-button .md-button--primary .mt-1 }

-   :material-account-hard-hat:{ .lg .middle } __Supporto professionale__

    ---

    Nextflow è un software gratuito e open-source sviluppato da [Seqera](https://seqera.io/), un'azienda con sede in Spagna e uffici satellite nel Regno Unito e negli Stati Uniti. Offriamo servizi di supporto professionale per Nextflow, inclusa formazione personalizzata.

    [Contattateci:material-arrow-right:](https://seqera.io/demo/){ .md-button .md-button--primary .mt-1 }

</div>
