## 1. I 3 modi principali per continuare il vostro percorso con i plugin

Ecco tre raccomandazioni su cosa fare dopo.

### 1.1. Esplorate il registro dei plugin

Sfogliate il [Nextflow Plugin Registry](https://registry.nextflow.io/) per vedere quali plugin sono disponibili.
Troverete plugin per la validazione degli input, l'integrazione con piattaforme cloud, le notifiche, il tracciamento della provenienza e molto altro.

### 1.2. Studiate il codice sorgente dei plugin esistenti

Esaminate il codice sorgente dei plugin più popolari per vedere come sono strutturati.
Il plugin [nf-hello](https://github.com/nextflow-io/nf-hello) è volutamente semplice e ben documentato.
Il [repository dei plugin ufficiali](https://github.com/nextflow-io/plugins) contiene esempi più complessi, tra cui executor e implementazioni di filesystem.

### 1.3. Costruite qualcosa di utile

Pensate a funzionalità che avreste voluto avere in Nextflow ma che non siete riusciti a trovare.
Alcune idee:

- Logging personalizzato o reportistica per la vostra organizzazione
- Integrazione con strumenti interni o API
- Funzioni di utilità specifiche per il vostro dominio
- Notifiche sulla vostra piattaforma preferita

Iniziate in piccolo, fatelo funzionare, poi iterate.
