# Nextflow Run

**Nextflow Run to praktyczne wprowadzenie do uruchamiania odtwarzalnych i skalowalnych workflow'ów analizy danych.**

Pracując nad praktycznymi przykładami i ćwiczeniami z przewodnikiem, nauczysz się podstaw używania Nextflow'a, w tym jak wykonywać pipeline'y, zarządzać plikami i zależnościami oprogramowania, bez wysiłku paralelizować wykonanie oraz uruchamiać workflow'y w różnych środowiskach obliczeniowych.

Wyniesiesz umiejętności i pewność siebie potrzebne do rozpoczęcia uruchamiania workflow'ów z Nextflow'em.

<!-- additional_information -->

## Przegląd kursu

### Co będziesz robić

Ten kurs jest praktyczny, z ćwiczeniami zorientowanymi na cel, ustrukturyzowanymi tak, aby stopniowo wprowadzać informacje.

Wykonasz kilka wersji pipeline'u Nextflow'a, który przetwarza dane tekstowe.
Zaczniesz od prostej wersji składającej się z jednego kroku, a ostatecznie przejdziesz do wersji wieloetapowej, która przyjmuje plik CSV z tabelarycznymi danymi tekstowymi, wykonuje kilka kroków transformacji i wyprowadza pojedynczy plik tekstowy zawierający obrazek ASCII postaci wypowiadającej przekształcony tekst.

Ten kurs koncentruje się na uruchamianiu pipeline'ów (nazwa pochodzi od podstawowego polecenia `nextflow run`).
Jeśli szukasz wprowadzenia do tworzenia pipeline'ów Nextflow'a, zobacz [Hello Nextflow](../hello_nextflow/index.md).
