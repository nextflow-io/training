## 1. Trzy najlepsze sposoby na kontynuowanie przygody z wtyczkami

Oto trzy rekomendacje dotyczące tego, co zrobić dalej.

### 1.1. Przeglądaj rejestr wtyczek

Zajrzyj do [Nextflow Plugin Registry](https://registry.nextflow.io/), aby zobaczyć, jakie wtyczki są dostępne.
Znajdziesz tam wtyczki do walidacji danych wejściowych, integracji z platformami chmurowymi, powiadomień, śledzenia proweniencji i nie tylko.

### 1.2. Studiuj kod źródłowy istniejących wtyczek

Przyjrzyj się kodowi źródłowemu popularnych wtyczek, aby zobaczyć, jak są zbudowane.
Wtyczka [nf-hello](https://github.com/nextflow-io/nf-hello) jest celowo prosta i dobrze udokumentowana.
[Oficjalne repozytorium wtyczek](https://github.com/nextflow-io/plugins) zawiera bardziej złożone przykłady, w tym implementacje executorów i systemów plików.

### 1.3. Zbuduj coś użytecznego

Zastanów się, jakiej funkcjonalności brakowało Ci w Nextflow, a której nie mogłeś znaleźć.
Kilka pomysłów:

- Niestandardowe logowanie lub raportowanie dla Twojej organizacji
- Integracja z wewnętrznymi narzędziami lub API
- Funkcje narzędziowe specyficzne dla danej dziedziny
- Powiadomienia na preferowaną przez Ciebie platformę

Zacznij od czegoś małego, doprowadź to do działania, a potem iteruj.
