# Nextflow dla Nauki

<span class="ai-translation-notice">:material-information-outline:{ .ai-translation-notice-icon } Tłumaczenie wspomagane przez AI - [dowiedz się więcej i zasugeruj ulepszenia](https://github.com/nextflow-io/training/blob/master/TRANSLATING.md)</span>

To są kursy, które pokazują, jak zastosować koncepcje i komponenty przedstawione w kursie dla początkujących [Hello Nextflow](../hello_nextflow/) do konkretnych przypadków użycia naukowego. Każdy kurs składa się z serii modułów szkoleniowych zaprojektowanych, aby pomóc osobom uczącym się stopniowo rozwijać ich umiejętności.

!!! exercise "Nextflow dla Genomiki"

    !!! tip inline end ""

        :material-run-fast: Naucz się opracowywać pipeline dla genomiki w Nextflow.

    To jest kurs dla badaczy, którzy chcą nauczyć się opracowywać własne pipeline'y genomiczne. Kurs wykorzystuje przypadek użycia wykrywania wariantów, aby zademonstrować opracowanie prostego, ale funkcjonalnego pipeline'u genomicznego.

    [Rozpocznij szkolenie Nextflow dla Genomiki :material-arrow-right:](genomics/){ .md-button .md-button--primary }

!!! exercise "Nextflow dla RNAseq"

    !!! tip inline end ""

        :material-run-fast: Naucz się opracowywać pipeline do przetwarzania danych RNAseq w Nextflow.

    To jest kurs dla badaczy, którzy chcą nauczyć się opracowywać własne pipeline'y RNAseq. Kurs wykorzystuje przypadek użycia przetwarzania bulk RNAseq, aby zademonstrować opracowanie prostego, ale funkcjonalnego pipeline'u RNAseq.

    [Rozpocznij szkolenie Nextflow dla RNAseq :material-arrow-right:](rnaseq/){ .md-button .md-button--primary }

!!! exercise "Nextflow dla Bioimaging"

    !!! tip inline end ""

        :material-run-fast: Naucz się uruchamiać pipeline'y do przetwarzania danych obrazowych w Nextflow.

    To jest kurs dla badaczy, którzy chcą nauczyć się uruchamiać i konfigurować pipeline'y bioimaging. Kurs wykorzystuje nf-core/molkart, aby zademonstrować podstawowe wzorce użycia Nextflow mające zastosowanie do każdego pipeline'u.

    [Rozpocznij szkolenie Nextflow dla Bioimaging :material-arrow-right:](imaging/){ .md-button .md-button--primary }
