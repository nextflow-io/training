# Nextflow dla nauki

To kursy, które pokazują, jak zastosować koncepcje i komponenty przedstawione w kursie dla początkujących [Hello Nextflow](../hello_nextflow/) do konkretnych przypadków użycia w nauce. Każdy kurs składa się z serii modułów szkoleniowych zaprojektowanych tak, aby pomóc uczestnikom stopniowo rozwijać swoje umiejętności.

!!! exercise "Nextflow dla genomiki"

    !!! tip inline end ""

        :material-run-fast: Naucz się tworzyć pipeline dla genomiki w Nextflow.

    To kurs dla naukowców, którzy chcą nauczyć się tworzyć własne pipeline'y genomiczne. Kurs wykorzystuje przypadek użycia wykrywania wariantów, aby pokazać, jak stworzyć prosty, ale funkcjonalny pipeline genomiczny.

    [Rozpocznij szkolenie Nextflow dla genomiki :material-arrow-right:](genomics/){ .md-button .md-button--primary }

!!! exercise "Nextflow dla RNAseq"

    !!! tip inline end ""

        :material-run-fast: Naucz się tworzyć pipeline do przetwarzania danych RNAseq w Nextflow.

    To kurs dla naukowców, którzy chcą nauczyć się tworzyć własne pipeline'y RNAseq. Kurs wykorzystuje przypadek użycia przetwarzania bulk RNAseq, aby pokazać, jak stworzyć prosty, ale funkcjonalny pipeline RNAseq.

    [Rozpocznij szkolenie Nextflow dla RNAseq :material-arrow-right:](rnaseq/){ .md-button .md-button--primary }

!!! exercise "Nextflow dla bioimagingu"

    !!! tip inline end ""

        :material-run-fast: Naucz się uruchamiać pipeline'y do przetwarzania danych obrazowych w Nextflow.

    To kurs dla naukowców, którzy chcą nauczyć się uruchamiać i konfigurować pipeline'y bioimagingowe. Kurs wykorzystuje nf-core/molkart, aby pokazać podstawowe wzorce użycia Nextflow'a, które można zastosować do dowolnego pipeline'u.

    [Rozpocznij szkolenie Nextflow dla bioimagingu :material-arrow-right:](imaging/){ .md-button .md-button--primary }

Daj nam znać, jakie inne dziedziny i przypadki użycia chciałbyś tutaj zobaczyć, publikując wpis w [sekcji szkoleniowej](https://community.seqera.io/c/training/) forum społeczności.
