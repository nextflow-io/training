# Uzyskiwanie pomocy

Niezależnie od tego, czy masz trudności z rozpoczęciem pracy, utknąłeś w połowie drogi, czy masz pytania uzupełniające, nie wahaj się skontaktować! Nasz zespół społeczności jest tutaj, aby pomóc, a społeczność Nextflow jest bardzo aktywna, otwarta i chętna do pomocy.

Poniżej znajdziesz główne opcje dostępne w zależności od tego, czego szukasz.

<div class="grid cards" markdown>

-   :material-forum-outline:{ .lg .middle } __Forum społeczności__

    ---

    Nasze forum społeczności ma kategorię dedykowaną szkoleniom, która jest świetnym miejscem do zadawania pytań lub zgłaszania problemów, które możesz napotkać podczas szkolenia. Możliwe, że Twoje pytanie zostało już zadane — i udzielono na nie odpowiedzi!

    [Dołącz do forum szkoleniowego:material-arrow-right:](https://community.seqera.io/c/training/39){ .md-button .md-button--primary .mt-1 }

-   :material-slack:{ .lg .middle } __Kanał Slack__

    ---

    Jeśli korzystasz ze Slacka, zapraszamy do kontaktu na kanale szkoleniowym w przestrzeni roboczej Nextflow Slack. Nextflow Slack to świetne miejsce do rozmów z innymi programistami i angażowania się w społeczność Nextflow w ogóle.

    [Dołącz do Nextflow Slack :material-arrow-right:](https://www.nextflow.io/slack-invite.html){ .md-button .md-button--primary .mt-1 }

-   :material-github:{ .lg .middle } __Zgłoszenia Github__

    ---

    Jeśli zauważysz błąd w materiałach szkoleniowych, zgłoś go, otwierając zgłoszenie w repozytorium Github. Niezależnie od tego, czy jest to literówka, problem z formatowaniem, czy faktyczny błąd wpływający na kod, daj nam znać, abyśmy mogli to naprawić! Przyjmujemy również bezpośrednio pull requesty.

    [Zgłoś problem:material-arrow-right:](https://github.com/nextflow-io/training/issues){ .md-button .md-button--primary .mt-1 }

-   :octicons-comment-ai-16:{ .lg .middle } __Asystent AI Seqera__

    ---

    Seqera AI to asystent AI wytrenowany na zasobach Nextflow i nf-core. Może pomóc Ci debugować kod, wyjaśniać koncepcje Nextflow i szybciej przeszukiwać dokumentację. Pomyśl o nim jak o tutorze dostępnym 24/7, gdy pracujesz nad kursami.

    [Zapytaj Seqera AI:material-arrow-right:](https://github.com/nextflow-io/training/issues){ .md-button .md-button--primary .mt-1 }

-   :material-book-open-variant:{ .lg .middle } __Dokumentacja Nextflow__

    ---

    Oficjalna dokumentacja to ostateczny przewodnik po wszystkich funkcjach języka i opcjach konfiguracji. Używaj jej wraz z tym szkoleniem, aby zgłębić konkretne tematy, poznać zaawansowane funkcje i znaleźć szczegółowe informacje o składni.

    [Przeglądaj dokumentację:material-arrow-right:](https://www.nextflow.io/docs/latest){ .md-button .md-button--primary .mt-1 }

-   :material-account-hard-hat:{ .lg .middle } __Wsparcie profesjonalne__

    ---

    Nextflow to darmowe oprogramowanie open-source rozwijane przez [Seqera](https://seqera.io/), firmę z siedzibą w Hiszpanii z biurami satelitarnymi w Wielkiej Brytanii i USA. Oferujemy profesjonalne usługi wsparcia dla Nextflow, w tym szkolenia dostosowane do potrzeb.

    [Skontaktuj się:material-arrow-right:](https://seqera.io/demo/){ .md-button .md-button--primary .mt-1 }
