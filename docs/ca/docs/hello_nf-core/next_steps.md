## El teu recorregut

Has començat aprenent a recuperar i executar un pipeline de demostració, després has abordat la conversió d'un workflow senzill de Nextflow en un pipeline nf-core.
Has après com crear una estructura de pipeline utilitzant una plantilla i has adaptat el pipeline existent a aquesta estructura.
Després has refinat gradualment el pipeline substituint un dels mòduls locals per un mòdul nf-core, has transformat un altre dels mòduls locals perquè complís els estàndards nf-core, i has afegit validació d'entrada.

### El que has construït

El teu pipeline final `core-hello` ara té:

- **Estructura estandarditzada** utilitzant la plantilla nf-core amb directoris organitzats per a workflows, subworkflows, mòduls i configuració
- **Mòduls de la comunitat** del repositori nf-core (`cat/cat`) juntament amb els teus mòduls personalitzats
- **Validació exhaustiva** que comprova tant els paràmetres com les dades d'entrada abans que s'executi el pipeline
- **Configuració professional** amb perfils per a diferents entorns d'execució
- **Documentació completa** i metadades seguint les convencions nf-core

### Habilitats clau adquirides

A través d'aquest curs pràctic, has après a:

1. **Navegar i comprendre** l'estructura dels pipelines nf-core explorant un pipeline existent
2. **Reestructurar workflows** perquè siguin composables i s'ajustin a la plantilla nf-core
3. **Trobar i integrar** mòduls preconstruïts del repositori de la comunitat
4. **Crear mòduls personalitzats** seguint els estàndards nf-core per a nomenclatura, estructura i metadades
5. **Implementar validació** utilitzant nf-schema per detectar errors de forma primerenca amb retroalimentació clara

Ara estàs equipat amb els coneixements fonamentals per construir pipelines nf-core preparats per a producció que segueixen les millors pràctiques de la comunitat.
