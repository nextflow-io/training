# Missions Secundàries

<span class="ai-translation-notice">:material-information-outline:{ .ai-translation-notice-icon } Traducció assistida per IA - [més informació i suggeriments](https://github.com/nextflow-io/training/blob/master/TRANSLATING.md)</span>

Aquesta és una col·lecció de mini-cursos de formació independents que aprofundeixen en temes específics. Podeu fer-los en qualsevol ordre.

Comencem! Feu clic al botó "Open in GitHub Codespaces" a continuació per llançar l'entorn de formació (preferiblement en una pestanya separada), i després continueu llegint mentre es carrega.

[![Open in GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/nextflow-io/training?quickstart=1&ref=master)

## Prerequisits

Els prerequisits específics de cada mini-curs varien i estan documentats a les pàgines corresponents.
No obstant això, tots assumeixen una familiaritat mínima amb el següent:

- Experiència amb la línia de comandes
- Conceptes i eines fonamentals de Nextflow coberts al curs de formació per a principiants [Hello Nextflow](../../hello_nextflow/).

Per als requisits tècnics i la configuració de l'entorn, consulteu el mini-curs [Configuració de l'entorn](../../envsetup/).

**Si és la primera vegada que us endinseu a les Missions Secundàries, assegureu-vos de consultar primer la pàgina d'[Orientació](./orientation.md)!**

En cas contrari, seleccioneu una missió secundària de la taula següent.
