## 1. Les 3 millors maneres de continuar el vostre camí amb els plugins

Aquí teniu tres recomanacions sobre què fer a continuació.

### 1.1. Exploreu el registre de plugins

Navegueu pel [Registre de Plugins de Nextflow](https://registry.nextflow.io/) per veure quins plugins estan disponibles.
Hi trobareu plugins per a la validació d'entrades, integració amb plataformes al núvol, notificacions, seguiment de procedència i molt més.

### 1.2. Estudieu el codi font de plugins existents

Mireu el codi font de plugins populars per veure com estan estructurats.
El plugin [nf-hello](https://github.com/nextflow-io/nf-hello) és intencionadament senzill i ben documentat.
El [repositori oficial de plugins](https://github.com/nextflow-io/plugins) conté exemples més complexos, incloent-hi executors i implementacions de sistemes de fitxers.

### 1.3. Construïu alguna cosa útil

Penseu en funcionalitats que hàgiu volgut tenir a Nextflow però que no hàgiu pogut trobar.
Algunes idees:

- Registre personalitzat o informes per a la vostra organització
- Integració amb eines internes o APIs
- Funcions utilitàries específiques del domini
- Notificacions a la vostra plataforma preferida

Comenceu amb quelcom petit, feu-ho funcionar i després itereu.

## 2. Obteniu ajuda de la comunitat

- [Nextflow Slack](https://www.nextflow.io/slack-invite.html): Feu preguntes i compartiu el vostre treball
- [Fòrum de la comunitat](https://community.seqera.io/): Debateu idees i obteniu consells
- [Discussions de GitHub](https://github.com/nextflow-io/nextflow/discussions): Preguntes tècniques i sol·licituds de funcionalitats

Si construïu un plugin útil, considereu compartir-lo amb la comunitat a través del registre de plugins.
