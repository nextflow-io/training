# Nextflow per a la Ciència

<span class="ai-translation-notice">:material-information-outline:{ .ai-translation-notice-icon } Traducció assistida per IA - [més informació i suggeriments](https://github.com/nextflow-io/training/blob/master/TRANSLATING.md)</span>

Aquests són cursos que demostren com aplicar els conceptes i components presentats al curs per a principiants [Hello Nextflow](../hello_nextflow/) a casos d'ús científics específics. Cada curs consisteix en una sèrie de mòduls de formació dissenyats per ajudar els estudiants a desenvolupar les seves habilitats progressivament.

!!! exercise "Nextflow per a Genòmica"

    !!! tip inline end ""

        :material-run-fast: Aprèn a desenvolupar un pipeline per a genòmica amb Nextflow.

    Aquest és un curs per a investigadors que volen aprendre a desenvolupar els seus propis pipelines de genòmica. El curs utilitza un cas d'ús de detecció de variants per demostrar com desenvolupar un pipeline de genòmica senzill però funcional.

    [Comença la formació de Nextflow per a Genòmica :material-arrow-right:](genomics/){ .md-button .md-button--primary }

!!! exercise "Nextflow per a RNAseq"

    !!! tip inline end ""

        :material-run-fast: Aprèn a desenvolupar un pipeline per al processament de dades RNAseq amb Nextflow.

    Aquest és un curs per a investigadors que volen aprendre a desenvolupar els seus propis pipelines de RNAseq. El curs utilitza un cas d'ús de processament de RNAseq massiu per demostrar com desenvolupar un pipeline de RNAseq senzill però funcional.

    [Comença la formació de Nextflow per a RNAseq :material-arrow-right:](rnaseq/){ .md-button .md-button--primary }

!!! exercise "Nextflow per a Bioimatge"

    !!! tip inline end ""

        :material-run-fast: Aprèn a executar pipelines per a dades d'imatge amb Nextflow.

    Aquest és un curs per a investigadors que volen aprendre a executar i configurar pipelines de bioimatge. El curs utilitza nf-core/molkart per demostrar patrons d'ús essencials de Nextflow aplicables a qualsevol pipeline.

    [Comença la formació de Nextflow per a Bioimatge :material-arrow-right:](imaging/){ .md-button .md-button--primary }

Fes-nos saber quins altres dominis i casos d'ús t'agradaria veure coberts aquí publicant a la [secció de Formació](https://community.seqera.io/c/training/) del fòrum de la comunitat.
