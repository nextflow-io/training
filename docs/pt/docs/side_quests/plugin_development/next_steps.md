## 1. As 3 principais formas de continuar sua jornada com plugins

Aqui estão três recomendações sobre o que fazer a seguir.

### 1.1. Explore o registro de plugins

Navegue pelo [Nextflow Plugin Registry](https://registry.nextflow.io/) para ver quais plugins estão disponíveis.
Você encontrará plugins para validação de entrada, integração com plataformas em nuvem, notificações, rastreamento de proveniência e muito mais.

### 1.2. Estude o código-fonte de plugins existentes

Veja o código-fonte de plugins populares para entender como eles são estruturados.
O plugin [nf-hello](https://github.com/nextflow-io/nf-hello) é intencionalmente simples e bem documentado.
O [repositório oficial de plugins](https://github.com/nextflow-io/plugins) contém exemplos mais complexos, incluindo executors e implementações de sistemas de arquivos.

### 1.3. Construa algo útil

Pense em funcionalidades que você sempre quis no Nextflow, mas não conseguiu encontrar.
Algumas ideias:

- Logging ou relatórios personalizados para sua organização
- Integração com ferramentas internas ou APIs
- Funções utilitárias específicas para seu domínio
- Notificações para sua plataforma preferida

Comece pequeno, faça funcionar e depois itere.
