# Missões Secundárias

<span class="ai-translation-notice">:material-information-outline:{ .ai-translation-notice-icon } Tradução assistida por IA - [saiba mais e sugira melhorias](https://github.com/nextflow-io/training/blob/master/TRANSLATING.md)</span>

**As Missões Secundárias são mini-cursos de treinamento independentes que se aprofundam em tópicos específicos do Nextflow.**

Cada missão secundária pode ser realizada de forma independente, em qualquer ordem, de acordo com seus interesses e necessidades.

## Visão geral do curso

Se esta é a primeira vez que você explora as Missões Secundárias, comece pela página de [Orientação](./orientation.md) para ter uma visão geral do ambiente de treinamento e dos materiais.
