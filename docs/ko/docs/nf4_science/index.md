# Nextflow for Science

과학 분야별 특정 사용 사례에 [Hello Nextflow](../hello_nextflow/) 초급 과정에서 다룬 개념과 구성 요소를 적용하는 방법을 보여주는 과정입니다. 각 과정은 학습자가 단계적으로 기술을 쌓을 수 있도록 설계된 일련의 교육 모듈로 구성되어 있습니다.

!!! exercise "Nextflow for Genomics"

    !!! tip inline end ""

        :material-run-fast: Nextflow로 유전체학 파이프라인을 개발하는 방법을 학습합니다.

    연구자가 자신만의 유전체학 파이프라인을 개발하는 방법을 학습하기 위한 과정입니다. 이 과정은 변이 검출 사용 사례를 통해 간단하지만 실용적인 유전체학 파이프라인을 개발하는 방법을 보여줍니다.

    [Nextflow for Genomics 교육 시작하기 :material-arrow-right:](genomics/){ .md-button .md-button--primary }
