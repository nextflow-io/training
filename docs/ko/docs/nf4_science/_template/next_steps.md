{DOMAIN} Nextflow 교육 과정을 완료하신 것을 축하합니다!

## 여러분의 여정

터미널에서 {METHOD} 도구를 수동으로 실행하여 방법론을 이해하는 것으로 시작했습니다.
그런 다음 프로세스를 자동화하기 위한 단일 샘플 Nextflow 파이프라인을 구축하고, 여러 샘플을 병렬로 처리하도록 확장했으며, 다중 샘플 {AGGREGATION_SUMMARY}를 추가했습니다.

### 구축한 내용

- {PRIMARY_INPUT_TYPE}을(를) 입력으로 받아 {FINAL_OUTPUT_TYPE}을(를) 출력으로 생성하는 {METHOD} 파이프라인.
- 별도의 모듈 파일에 저장된 {N}개의 프로세스({PROCESS_LIST}).
- Nextflow의 데이터플로우 패러다임을 사용하여 입력 샘플 수에 관계없이 자동으로 확장되는 파이프라인.
- `results/` 디렉토리에 게시되는 결과.

### 습득한 기술

이 실습 과정을 통해 다음 방법을 학습했습니다:

- 단일 샘플에 {METHOD}을(를) 적용하는 선형 워크플로우 작성
- {ACCESSORY_FILES}와 같은 보조 파일을 적절하게 처리
- Nextflow의 데이터플로우 패러다임을 활용하여 샘플별 처리 병렬화
- 관련 채널 연산자를 사용한 다중 샘플 집계 구현

이제 여러분은 자신의 작업에서 {DOMAIN} 분석 워크플로우에 Nextflow를 적용할 준비가 되었습니다.
