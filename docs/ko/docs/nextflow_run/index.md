# Nextflow Run

**Nextflow Run은 재현 가능하고 확장 가능한 데이터 분석 워크플로우를 실행하는 실습 입문 과정입니다.**

실용적인 예제와 안내된 연습을 통해 파이프라인 실행, 파일 및 소프트웨어 의존성 관리, 손쉬운 병렬 실행, 다양한 컴퓨팅 환경에서의 워크플로우 실행 방법을 포함하여 Nextflow 사용의 기초를 학습합니다.

Nextflow로 워크플로우를 실행하기 시작할 수 있는 기술과 자신감을 얻게 됩니다.

<!-- additional_information -->

## 과정 개요

### 학습 내용

이 과정은 실습 중심이며, 정보를 단계적으로 소개하도록 구성된 목표 지향적 연습으로 이루어져 있습니다.

텍스트 입력을 처리하는 여러 버전의 Nextflow 파이프라인을 실행합니다.
단일 단계로 구성된 간단한 버전부터 시작하여, 결국 CSV 파일의 표 형식 텍스트 입력을 받아 몇 가지 변환 단계를 실행하고 변환된 텍스트를 말하는 캐릭터의 ASCII 그림이 포함된 단일 텍스트 파일을 출력하는 다단계 버전으로 진행합니다.

이 과정은 파이프라인 실행에 중점을 둡니다(핵심 `nextflow run` 명령의 이름을 따서 명명됨).
Nextflow 파이프라인 개발 입문을 찾고 있다면 [Hello Nextflow](../hello_nextflow/index.md)를 참조하세요.
