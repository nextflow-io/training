# Hello Nextflow

**Hello Nextflow는 재현 가능하고 확장 가능한 데이터 분석 워크플로우를 구축하는 실습 중심 입문 과정입니다.**

실용적인 예제와 안내된 연습을 통해 프로세스 정의, 파이프라인 연결, 파일 및 소프트웨어 의존성 관리, 손쉬운 병렬 실행, 다양한 컴퓨팅 환경에서의 워크플로우 실행 등 Nextflow로 파이프라인을 개발하는 기본 원리를 학습합니다.

이 과정을 마치면 Nextflow로 자신만의 워크플로우를 개발하고 실행할 수 있는 기술과 자신감을 얻게 됩니다.

<!-- additional_information -->

## 과정 개요

이 과정은 실습 중심으로 설계되었으며, 정보를 단계적으로 소개하도록 구성된 목표 지향적 연습으로 이루어져 있습니다.

텍스트 입력을 받아 몇 가지 변환 단계를 거쳐 변환된 텍스트를 말하는 캐릭터의 ASCII 그림이 포함된 단일 텍스트 파일을 출력하는 간단한 Nextflow 파이프라인을 개발합니다.

### 학습 계획

개념과 코드로 인한 부담을 피하기 위해, Nextflow로 파이프라인을 개발하는 특정 측면에 각각 초점을 맞춘 6개 파트로 나누었습니다.

| 과정 챕터                                            | 요약                                                                      | 예상 소요 시간 |
| ---------------------------------------------------- | ------------------------------------------------------------------------- | -------------- |
| [Part 1: Hello World](./01_hello_world.md)           | Nextflow 워크플로우를 구성하고 실행하는 데 필요한 기본 구성 요소 및 원리 | 30분           |
| [Part 2: Hello Channels](./02_hello_channels.md)     | 채널과 연산자를 사용하여 입력을 처리하고 손쉽게 병렬 실행하기             | 45분           |
| [Part 3: Hello Workflow](./03_hello_workflow.md)     | 채널을 사용하여 여러 단계를 연결하고 단계 간 데이터 전송 처리하기        | 60분           |
| [Part 4: Hello Modules](./04_hello_modules.md)       | 코드 모듈화 원칙을 적용하여 재사용성을 높이고 유지보수 부담 줄이기       | 20분           |
| [Part 5: Hello Containers](./05_hello_containers.md) | 컨테이너를 사용하여 소프트웨어 의존성을 관리하고 재현성 향상시키기       | 60분           |
| [Part 6: Hello Config](./06_hello_config.md)         | 파이프라인 동작을 사용자 정의하고 다양한 컴퓨팅 환경에서 사용 최적화하기 | 60분           |

이 과정을 마치면 과학 컴퓨팅 요구 사항에 맞는 재현 가능한 워크플로우를 개발하는 다음 단계를 준비할 수 있습니다.
