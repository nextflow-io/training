# 교육 컬렉션

<span class="ai-translation-notice">:material-information-outline:{ .ai-translation-notice-icon } AI 지원 번역 - [자세히 알아보기 및 개선 사항 제안](https://github.com/nextflow-io/training/blob/master/TRANSLATING.md)</span>

이 섹션에는 특정 주제나 사용 사례를 중심으로 포괄적인 학습 경험을 제공하는 것을 목표로 하는 [Side Quests](../side_quests/index.md)라는 교육 모듈의 선별된 컬렉션이 포함되어 있습니다.

## 사전 요구 사항

각 컬렉션에는 인덱스 페이지에 문서화된 특정 사전 요구 사항이 있습니다. 그러나 대부분의 컬렉션은 다음을 가정합니다:

- 명령줄 사용 경험
- [Hello Nextflow](../../hello_nextflow/) 초급 교육 과정에서 다루는 기본적인 Nextflow 개념 및 도구

기술적 요구 사항 및 환경 설정에 대해서는 [환경 설정](../../envsetup/) 단기 과정을 참조하세요.

## 사용 가능한 컬렉션

- [The Architect's Toolkit I](./architects_toolkit_1.md) - 복잡한 파이프라인 조립을 위한 워크플로우 아키텍처 패턴, 테스트 전략 구현, 메타데이터 관리, 데이터 그룹화 및 분할을 다루는 4개의 Side Quests 컬렉션입니다. _예상 소요 시간: 그룹 교육 기준 4시간._

## 새로운 컬렉션 제안하기

저희는 추가 Side Quests 및 컬렉션 개발을 적극적으로 진행하고 있습니다.
커뮤니티 포럼의 [교육 섹션](https://community.seqera.io/c/training/)에 게시하여 컬렉션에서 다루면 좋을 것 같은 주제를 자유롭게 제안해 주세요.
