# 도움 받기

시작하는 데 어려움을 겪고 있거나, 중간에 막히거나, 후속 질문이 있다면 주저하지 말고 연락하세요! 커뮤니티 팀이 도와드리며, Nextflow 커뮤니티는 매우 활발하고 포용적이며 기꺼이 도움을 제공합니다.

찾고 계신 내용에 따라 다음과 같은 주요 옵션을 이용하실 수 있습니다.

<div class="grid cards" markdown>

-   :material-forum-outline:{ .lg .middle } __커뮤니티 포럼__

    ---

    커뮤니티 포럼에는 교육 전용 카테고리가 있으며, 질문을 게시하거나 교육 중 발생한 문제를 보고하기에 좋은 곳입니다. 이미 질문이 올라와 답변을 받았을 수도 있습니다!

    [교육 포럼 참여하기:material-arrow-right:](https://community.seqera.io/c/training/39){ .md-button .md-button--primary .mt-1 }

-   :material-slack:{ .lg .middle } __Slack 채널__

    ---

    Slack을 사용하신다면, Nextflow Slack 워크스페이스의 교육 채널에서 언제든지 연락하실 수 있습니다. Nextflow Slack은 동료 개발자들과 대화하고 Nextflow 커뮤니티와 전반적으로 소통할 수 있는 좋은 공간입니다.

    [Nextflow Slack 참여하기 :material-arrow-right:](https://www.nextflow.io/slack-invite.html){ .md-button .md-button--primary .mt-1 }

-   :material-github:{ .lg .middle } __Github 이슈__

    ---

    교육 자료에서 오류를 발견하셨다면, Github 저장소에 이슈를 열어 보고해 주세요. 오타, 서식 문제 또는 코드에 영향을 미치는 실제 버그 등 무엇이든 알려주시면 수정하겠습니다! PR도 환영합니다.

    [이슈 보고하기:material-arrow-right:](https://github.com/nextflow-io/training/issues){ .md-button .md-button--primary .mt-1 }

-   :octicons-comment-ai-16:{ .lg .middle } __Seqera AI 어시스턴트__

    ---

    Seqera AI는 Nextflow 및 nf-core 리소스로 학습된 AI 어시스턴트입니다. 코드 디버깅, Nextflow 개념 설명, 문서 검색을 더 빠르게 도와줍니다. 과정을 진행하는 동안 24시간 내내 이용 가능한 튜터가 있다고 생각하시면 됩니다.

    [Seqera AI에게 질문하기:material-arrow-right:](https://github.com/nextflow-io/training/issues){ .md-button .md-button--primary .mt-1 }

-   :material-book-open-variant:{ .lg .middle } __Nextflow 문서__

    ---

    공식 문서는 모든 언어 기능과 구성 옵션에 대한 최고의 가이드입니다. 이 교육과 함께 사용하여 특정 주제를 더 깊이 탐구하고, 고급 기능을 살펴보고, 상세한 구문 참조를 찾아보세요.

    [문서 둘러보기:material-arrow-right:](https://www.nextflow.io/docs/latest){ .md-button .md-button--primary .mt-1 }

-   :material-account-hard-hat:{ .lg .middle } __전문 지원__

    ---

    Nextflow는 스페인에 본사를 두고 영국과 미국에 지사를 둔 [Seqera](https://seqera.io/)에서 개발한 무료 오픈 소스 소프트웨어입니다. 맞춤형 교육을 포함한 Nextflow 전문 지원 서비스를 제공합니다.

    [문의하기:material-arrow-right:](https://seqera.io/demo/){ .md-button .md-button--primary .mt-1 }
