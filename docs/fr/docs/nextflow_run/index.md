# Nextflow Run

**Nextflow Run est une introduction pratique à l'exécution de workflows d'analyse de données reproductibles et évolutifs.**

En travaillant sur des exemples pratiques et des exercices guidés, vous apprendrez les fondamentaux de l'utilisation de Nextflow, notamment comment exécuter des pipelines, gérer les fichiers et les dépendances logicielles, paralléliser l'exécution sans effort, et exécuter des workflows dans différents environnements informatiques.

Vous acquerrez les compétences et la confiance nécessaires pour commencer à exécuter des workflows avec Nextflow.

<!-- additional_information -->

## Aperçu du cours

### Ce que vous allez faire

Ce cours est pratique, avec des exercices orientés vers des objectifs structurés pour introduire l'information progressivement.

Vous exécuterez plusieurs versions d'un pipeline Nextflow qui traite des entrées textuelles.
Vous commencerez par une version simple qui consiste en une seule étape, et progresserez éventuellement vers une version multi-étapes qui prend un fichier CSV d'entrées textuelles tabulaires, exécute quelques étapes de transformation, et produit un seul fichier texte contenant une image ASCII d'un personnage disant le texte transformé.

Ce cours se concentre sur l'exécution de pipelines (nommé d'après la commande principale `nextflow run`).
Si vous cherchez une introduction au développement de pipelines Nextflow, consultez [Hello Nextflow](../hello_nextflow/index.md).

### Plan de cours

Nous avons divisé cela en trois parties qui se concentreront chacune sur des aspects spécifiques de l'exécution et de la gestion de pipelines écrits en Nextflow.

| Chapitre du cours                                      | Résumé                                                                                                                                                  | Durée estimée |
| ------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------- |
| [Partie 1 : Opérations de base](./01_basics.md)        | Lancement et gestion de l'exécution d'un workflow simple                                                                                                | 30 min        |
| [Partie 2 : Exécuter de vrais pipelines](./02_pipeline.md) | Traitement d'entrées complexes, exécution de workflows multi-étapes, utilisation de conteneurs et parallélisation de l'exécution sans effort            | 60 min        |
| [Partie 3 : Configuration d'exécution](./03_config.md) | Personnalisation du comportement du pipeline et optimisation de l'utilisation dans différents environnements informatiques                              | 60 min        |

À la fin de ce cours, vous serez bien préparé·e pour aborder les prochaines étapes de votre parcours vers l'exécution de workflows reproductibles pour vos besoins en calcul scientifique.
