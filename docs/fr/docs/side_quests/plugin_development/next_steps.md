## 1. Les 3 meilleures façons de poursuivre votre parcours avec les plugins

Voici trois recommandations pour la suite.

### 1.1. Explorer le registre de plugins

Parcourez le [Nextflow Plugin Registry](https://registry.nextflow.io/) pour découvrir les plugins disponibles.
Vous y trouverez des plugins pour la validation des entrées, l'intégration avec des plateformes cloud, les notifications, le suivi de provenance, et bien plus encore.

### 1.2. Étudier le code source de plugins existants

Examinez le code source de plugins populaires pour comprendre leur structure.
Le plugin [nf-hello](https://github.com/nextflow-io/nf-hello) est intentionnellement simple et bien documenté.
Le [dépôt officiel des plugins](https://github.com/nextflow-io/plugins) contient des exemples plus complexes, notamment des executors et des implémentations de systèmes de fichiers.

### 1.3. Créer quelque chose d'utile

Réfléchissez aux fonctionnalités que vous avez souhaitées dans Nextflow sans pouvoir les trouver.
Quelques idées :

- Journalisation ou rapports personnalisés pour votre organisation
- Intégration avec des outils internes ou des API
- Fonctions utilitaires spécifiques à votre domaine
- Notifications vers votre plateforme préférée

Commencez petit, faites-le fonctionner, puis itérez.
