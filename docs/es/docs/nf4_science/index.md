# Nextflow para Ciencia

Estos son cursos que demuestran cómo aplicar los conceptos y componentes presentados en el curso para principiantes [Hello Nextflow](../hello_nextflow/) a casos de uso científicos específicos. Cada curso consiste en una serie de módulos de capacitación diseñados para ayudar a los estudiantes a desarrollar sus habilidades de manera progresiva.

!!! exercise "Nextflow para Genómica"

    !!! tip inline end ""

        :material-run-fast: Aprende a desarrollar un pipeline para genómica en Nextflow.

    Este es un curso para investigadores que desean aprender a desarrollar sus propios pipelines de genómica. El curso utiliza un caso de uso de llamado de variantes para demostrar cómo desarrollar un pipeline de genómica simple pero funcional.

    [Comenzar la capacitación de Nextflow para Genómica :material-arrow-right:](genomics/){ .md-button .md-button--primary }

!!! exercise "Nextflow para RNAseq"

    !!! tip inline end ""

        :material-run-fast: Aprende a desarrollar un pipeline para procesamiento de datos RNAseq en Nextflow.

    Este es un curso para investigadores que desean aprender a desarrollar sus propios pipelines de RNAseq. El curso utiliza un caso de uso de procesamiento de RNAseq masivo para demostrar cómo desarrollar un pipeline de RNAseq simple pero funcional.

    [Comenzar la capacitación de Nextflow para RNAseq :material-arrow-right:](rnaseq/){ .md-button .md-button--primary }

!!! exercise "Nextflow para Bioimagen"

    !!! tip inline end ""

        :material-run-fast: Aprende a ejecutar pipelines para datos de imágenes en Nextflow.

    Este es un curso para investigadores que desean aprender a ejecutar y configurar pipelines de bioimagen. El curso utiliza nf-core/molkart para demostrar patrones esenciales de uso de Nextflow aplicables a cualquier pipeline.

    [Comenzar la capacitación de Nextflow para Bioimagen :material-arrow-right:](imaging/){ .md-button .md-button--primary }

Cuéntanos qué otros dominios y casos de uso te gustaría ver cubiertos aquí publicando en la [sección de Capacitación](https://community.seqera.io/c/training/) del foro de la comunidad.
