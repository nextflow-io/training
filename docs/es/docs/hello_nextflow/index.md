# Hello Nextflow

**Hello Nextflow es una introducción práctica a la construcción de flujos de trabajo de análisis de datos reproducibles y escalables.**

A través de ejemplos prácticos y ejercicios guiados, aprenderás los fundamentos del desarrollo de pipelines con Nextflow, incluyendo cómo definir procesos, conectarlos en pipelines, gestionar archivos y dependencias de software, paralelizar la ejecución sin esfuerzo y ejecutar workflows en diferentes entornos computacionales.

Adquirirás las habilidades y la confianza para comenzar a desarrollar y ejecutar tus propios workflows con Nextflow.

<!-- additional_information -->

## Descripción general del curso

Este curso está diseñado para ser práctico, con ejercicios orientados a objetivos estructurados para introducir información gradualmente.

Desarrollarás un pipeline simple de Nextflow que toma algunas entradas de texto, ejecuta algunos pasos de transformación y genera un único archivo de texto que contiene una imagen ASCII de un personaje diciendo el texto transformado.

### Plan de lecciones

Para evitar abrumarte con conceptos y código, hemos dividido esto en seis partes que se enfocarán en aspectos específicos del desarrollo de pipelines con Nextflow.

| Capítulo del curso                                   | Resumen                                                                                                                | Duración estimada |
| ---------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------- | ----------------- |
| [Parte 1: Hello World](./01_hello_world.md)          | Componentes básicos y principios involucrados en ensamblar y ejecutar un workflow de Nextflow                         | 30 mins           |
| [Parte 2: Hello Channels](./02_hello_channels.md)   | Uso de canales y operadores para procesar entradas y paralelizar la ejecución sin esfuerzo                            | 45 mins           |
| [Parte 3: Hello Workflow](./03_hello_workflow.md)   | Uso de canales para encadenar múltiples pasos y manejar la transferencia de datos entre pasos                         | 60 mins           |
| [Parte 4: Hello Modules](./04_hello_modules.md)     | Aplicación de principios de modularidad de código para aumentar la reutilización y disminuir la carga de mantenimiento | 20 mins           |
| [Parte 5: Hello Containers](./05_hello_containers.md) | Uso de contenedores como mecanismo para gestionar dependencias de software y aumentar la reproducibilidad             | 60 mins           |
| [Parte 6: Hello Config](./06_hello_config.md)       | Personalización del comportamiento del pipeline y optimización del uso en diferentes entornos computacionales         | 60 mins           |

Al final de este curso, estarás bien preparado/a para abordar los siguientes pasos en tu camino hacia el desarrollo de workflows reproducibles para tus necesidades de computación científica.
