# Nextflow Run

**Nextflow Run es una introducción práctica a la ejecución de flujos de trabajo de análisis de datos reproducibles y escalables.**

A través de ejemplos prácticos y ejercicios guiados, aprenderás los fundamentos del uso de Nextflow, incluyendo cómo ejecutar pipelines, gestionar archivos y dependencias de software, paralelizar la ejecución sin esfuerzo y ejecutar workflows en diferentes entornos computacionales.

Adquirirás las habilidades y la confianza para comenzar a ejecutar workflows con Nextflow.

<!-- additional_information -->

## Descripción del curso

### Qué harás

Este curso es práctico, con ejercicios orientados a objetivos estructurados para introducir información gradualmente.

Ejecutarás varias versiones de un pipeline de Nextflow que procesa entradas de texto.
Comenzarás con una versión simple que consiste en un solo paso, y eventualmente progresarás a una versión de múltiples pasos que toma un archivo CSV de entradas de texto tabulares, ejecuta algunos pasos de transformación y genera un único archivo de texto que contiene una imagen ASCII de un personaje diciendo el texto transformado.

Este curso se enfoca en ejecutar pipelines (nombrado así por el comando principal `nextflow run`).
Si buscas una introducción al desarrollo de pipelines de Nextflow, consulta [Hello Nextflow](../hello_nextflow/index.md).

### Plan de lecciones

Hemos dividido esto en tres partes que se enfocarán en aspectos específicos de la ejecución y gestión de pipelines escritos en Nextflow.

| [Parte 2: Ejecutar pipelines reales](./02_pipeline.md) | Procesar entradas complejas, ejecutar workflows de múltiples pasos, usar contenedores y paralelizar la ejecución sin esfuerzo           | 60 mins           |

Al final de este curso, estarás bien preparado/a para abordar los siguientes pasos en tu camino hacia la ejecución de workflows reproducibles para tus necesidades de computación científica.
