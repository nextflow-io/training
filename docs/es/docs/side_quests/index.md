# Side Quests

<span class="ai-translation-notice">:material-information-outline:{ .ai-translation-notice-icon } Traducción asistida por IA - [más información y sugerencias](https://github.com/nextflow-io/training/blob/master/TRANSLATING.md)</span>

**Las Side Quests son mini-cursos de entrenamiento independientes que profundizan en temas específicos de Nextflow.**

Cada Side Quest puede tomarse de forma independiente, en cualquier orden, según sus intereses y necesidades.

<!-- additional_information -->

## Descripción general del curso

Si esta es la primera vez que explora las Side Quests, comience con la página de [Orientación](./orientation.md) para obtener una descripción general del entorno de entrenamiento y los materiales.

### Side Quests

| Side Quest                                                                  | Resumen                                                                          | Tiempo Estimado |
| --------------------------------------------------------------------------- | -------------------------------------------------------------------------------- | --------------- |
| [Entorno de Desarrollo](./dev_environment/index.md)                         | Configure un entorno de desarrollo local de Nextflow productivo                  | 45 mins         |
| [Patrones Esenciales de Scripting](./essential_scripting_patterns/index.md) | Técnicas avanzadas de scripting para desafíos comunes en workflows               | 90 mins         |
| [Procesamiento de Archivos de Entrada](./working_with_files/index.md)       | Manejo de archivos, operaciones de rutas y organización de salidas               | 45 mins         |
| [Metadatos y Meta Maps](./metadata/index.md)                                | Uso de mapas de metadatos para rastrear y propagar información de muestras       | 45 mins         |
| [División y Agrupación](./splitting_and_grouping/index.md)                  | Técnicas para dividir y reagrupar canales de datos                               | 45 mins         |
| [Pruebas con nf-test](./nf_test/index.md)                                   | Escritura y ejecución de pruebas para workflows de Nextflow                      | 1 hora          |
| [Depuración de Workflows](./debugging/index.md)                             | Identificación y corrección de errores comunes en workflows                      | 1 hora          |
| [Workflows de Workflows](./workflows_of_workflows/index.md)                 | Composición de pipelines complejos a partir de módulos de workflow reutilizables | 30 mins         |
| [Desarrollo de Plugins](./plugin_development/index.md)                      | Uso y creación de plugins de Nextflow                                            | 3 horas         |
