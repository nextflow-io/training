## 1. Las 3 mejores formas de continuar su camino con los plugins

Aquí hay tres recomendaciones sobre qué hacer a continuación.

### 1.1. Explore el registro de plugins

Navegue por el [Registro de Plugins de Nextflow](https://registry.nextflow.io/) para ver qué plugins están disponibles.
Encontrará plugins para validación de entradas, integración con plataformas en la nube, notificaciones, seguimiento de procedencia y más.

### 1.2. Estudie el código fuente de plugins existentes

Examine el código fuente de plugins populares para ver cómo están estructurados.
El plugin [nf-hello](https://github.com/nextflow-io/nf-hello) es intencionalmente simple y está bien documentado.
El [repositorio oficial de plugins](https://github.com/nextflow-io/plugins) contiene ejemplos más complejos, incluyendo executors e implementaciones de sistemas de archivos.

### 1.3. Construya algo útil

Piense en funcionalidades que haya deseado tener en Nextflow pero que no haya podido encontrar.
Algunas ideas:

- Registro o reportes personalizados para su organización
- Integración con herramientas internas o APIs
- Funciones utilitarias específicas para su dominio
- Notificaciones a su plataforma preferida

Empiece con algo pequeño, hágalo funcionar y luego itere.
