    Side Quests¶

    This is a collection of standalone training mini-courses that go deeper into specific topics. You can go through them in any order.

    Let's get started! Click on the "Open in GitHub Codespaces" button below to launch the training environment (preferably in a separate tab), then read on while it loads.

    Prerequisites¶

    The specific prerequisites of each mini-courses vary and are documented in the corresponding pages. Nevertheless, they all assume some minimal familiarity with the following:

    • Experience with the command line
    • Foundational Nextflow concepts and tooling covered in the Hello Nextflow beginner training course.

    For technical requirements and environment setup, see the Environment Setup mini-course.

    If this is the first time you delve into the Side Quests, make sure to check out the Orientation page first!

    Otherwise, select a side quest from the menu below.
